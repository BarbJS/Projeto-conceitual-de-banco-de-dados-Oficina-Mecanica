# Projeto-conceitual-de-banco-de-dados-Oficina-Mecanica
Este repositório contém o modelo conceitual e lógico de banco de dados para um Sistema de Gerenciamento de Oficina Mecânica. O projeto foi desenhado para atender a requisitos complexos de ordens de serviço, gestão de equipes e precificação dinâmica de mão-de-obra de serviços e peças.
